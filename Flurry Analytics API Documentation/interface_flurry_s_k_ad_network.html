+ (void) flurryUpdateConversionValueWithEvent:

Call this api to allow Flurry to set your conversion value. The final conversion value is a decimal number between 0-63. The conversion value is calculated from a 6 bit binary number. The first two bits represent days of user retention from 0-3 days The last four bits represent a true false state indicating if the user has completed the post install event.
